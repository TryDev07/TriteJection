# TriteJection

___
Dependency injection for java.

###

maven repository:

```xml

<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>

<dependencies>
    <dependency>
	    <groupId>com.github.TryDev07</groupId>
	    <artifactId>TriteJection</artifactId>
	    <version>v1.2.0</version>
	</dependency>
</dependencies>
```
